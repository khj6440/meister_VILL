혁진



1. 관리자 마이페이지

	- 회원관리 ( 검색 수정 삭제 )
	- 판매중 리스트 확인
	- 후기리스트 확인
	- 신고댓글이나 후기 별도 확인
	- 리스트 정렬 및 검색

2. 구매자 마이페이지
	
	- 후기작성 팝업(일반후기,포토후기)
	- 구매자 정보 수정 및 탈퇴
	- 구매참여, 구매완료 목록 확인
	- 구매 확정기능 부여

3. 판매자 마이페이지

	- 판매중, 판매완료 목록 확인
	- 판매자 정보 수정 및 탈퇴
	- 리스트 정렬 및 검색
